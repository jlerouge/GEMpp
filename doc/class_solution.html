The Solution class represents a feasible solution of a formulation.

Member Enumeration Documentation

enum Solution::Status

Indicates the status of the solution, as returned by the solver.

Enumerator
NOT_SOLVED 

the problem was not solved

INFEASIBLE 

the problem is infeasible

UNBOUNDED 

the problem is unbounded (the best solution is infinite)

SUBOPTIMAL 

best solution found after early-stopping the problem solving

OPTIMAL 

best possible solution

COUNT 

used to iterate on Solution::Status enum
