Detailed Description

The GraphElement class represents an abstract element of a Graph, i.e. a Vertex or an Edge. These elements have attributes, and it is possible to compute substitution costs between two GraphElement of the same type as well as the creation cost of a single GraphElement, using the substitution and creation Weights.
