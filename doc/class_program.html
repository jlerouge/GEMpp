Detailed Description

The Program class represents the mathematical concept of integer programs, that are used to solve integer optimization problems.

Let be a vector of integer variables and let be a function, named the objective function.

An integer Program is used to solve an integer optimization problem, that is to say optimizing an objective function over a discrete set, under certain conditions on the integer variables, called the constraints. A Constraint can be either an equality constraint or an inequality constraint.
