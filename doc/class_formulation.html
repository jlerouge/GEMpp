Detailed Description

The Formulation class represents the mathematical programming formulation used to solve a graph matching problem.

The formulation is dedicated to a particular problem (subgraph isomorphism, graph edit distance...), an thus generates a linear/quadratic program that can be used to solve it.

Author

Warning
This is an abstract class, it must be derived to be used.

enum Formulation::CutMethod

The method used to cut a previously found solution.

Enumerator
SOLUTION 

makes only this particular solution no more feasible

MATCHINGS 

makes every vertex matching and every edge matching no more feasible

ELEMENTS 

makes every vertex matched and every edge matched no more matchable

COUNT 

used to iterate on Formulation::CutMethod enum
